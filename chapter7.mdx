# Chapter 7: Data Generation, Conversion, and Manipulation

## String Data

### Character Data Types

- CHAR: Fixed-length strings
- VARCHAR: Variable-length strings
- TEXT/CLOB: Large text data

### String Generation

```sql
INSERT INTO string_tbl (char_fld, vchar_fld, text_fld)
VALUES ('This is char data',
        'This is varchar data',
        'This is text data');
```

### Handling Special Characters

Use double single quotes or backslashes to escape single quotes:

```sql
UPDATE string_tbl
SET text_fld = 'This string didn''t work, but it does now';

UPDATE string_tbl
SET text_fld = 'This string didn\'t work, but it does now';
```

### Including Special Characters

Use the CHAR() function:

```sql
SELECT CHAR(128,129,130,131,132,133,134,135,136,137);
```

## String Manipulation

### String Functions Returning Numbers

LENGTH() function:

```sql
SELECT LENGTH(char_fld) char_length,
       LENGTH(vchar_fld) varchar_length,
       LENGTH(text_fld) text_length
FROM string_tbl;
```

POSITION() function:

```sql
SELECT POSITION('characters' IN vchar_fld)
FROM string_tbl;
```

### String Functions Returning Strings

CONCAT() function:

```sql
SELECT CONCAT(fname, ' ', lname, ' has been a ',
              title, ' since ', start_date) emp_narrative
FROM employee
WHERE title = 'Teller' OR title = 'Head Teller';
```

SUBSTRING() function:

```sql
SELECT SUBSTRING('goodbye cruel world', 9, 5);
```

## Numeric Data

### Numeric Data Types

- TINYINT, SMALLINT, MEDIUMINT, INT, BIGINT: Integer types
- FLOAT, DOUBLE: Floating-point types

### Numeric Functions

POW() function:

```sql
SELECT POW(2,8);
```

### Controlling Number Precision

ROUND() function:

```sql
SELECT ROUND(72.0909, 1), ROUND(72.0909, 2), ROUND(72.0909, 3);
```

TRUNCATE() function:

```sql
SELECT TRUNCATE(72.0909, 1), TRUNCATE(72.0909, 2),
       TRUNCATE(72.0909, 3);
```

## Temporal Data

### Temporal Data Types

- DATE: Date only
- DATETIME: Date and time
- TIME: Time only
- TIMESTAMP: Date and time with time zone awareness

### Generating Temporal Data

Using string representations:

```sql
UPDATE transaction
SET txn_date = '2008-09-17 15:30:00'
WHERE txn_id = 99999;
```

Using built-in functions:

```sql
SELECT CURRENT_DATE(), CURRENT_TIME(), CURRENT_TIMESTAMP();
```

### Manipulating Temporal Data

DATE_ADD() function:

```sql
SELECT DATE_ADD(CURRENT_DATE(), INTERVAL 5 DAY);
```

DATEDIFF() function:

```sql
SELECT DATEDIFF('2009-09-03', '2009-06-24');
```

## Conversion Functions

CAST() function:

```sql
SELECT CAST('1456328' AS SIGNED INTEGER);
```

STR_TO_DATE() function:

```sql
SELECT STR_TO_DATE('September 17, 2008', '%M %d, %Y');
```

## Key Takeaways

- String manipulation functions are essential for data cleaning and formatting
- Numeric functions allow for complex calculations within queries
- Temporal data requires special handling for storage and manipulation
- Conversion functions are crucial for ensuring data type compatibility
- Understanding these functions enables complex data transformations within SQL

Remember that while these examples use MySQL syntax, other databases may have slightly different function names or syntax for similar operations.
