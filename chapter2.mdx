````mdx
# Chapter 2: Creating and Populating a Database

## 1. Database Creation

- Instructions for installing MySQL version 6.0 or later
- Creating a database:

```sql
CREATE DATABASE bank;
```
````

- Creating a user and granting privileges:

```sql
GRANT ALL PRIVILEGES ON bank.* TO 'username'@'localhost' IDENTIFIED BY 'password';
```

## 2. MySQL Data Types

### a. Character Data

- `CHAR`: Fixed-length strings (up to 255 characters in MySQL)
- `VARCHAR`: Variable-length strings (up to 65,535 characters in MySQL)
- `TEXT`: For larger strings (documents)

### b. Numeric Data

- Integer types: `TINYINT`, `SMALLINT`, `MEDIUMINT`, `INT`, `BIGINT`
- Floating-point types: `FLOAT(p,s)`, `DOUBLE(p,s)`
  - Where 'p' is precision (total digits) and 's' is scale (decimal places)

### c. Temporal Data

- `DATE`: YYYY-MM-DD format
- `DATETIME`: YYYY-MM-DD HH:MI:SS
- `TIMESTAMP`: Similar to DATETIME but with automatic updating
- `YEAR`: YYYY
- `TIME`: HHH:MI:SS

## 3. Table Creation

- CREATE TABLE syntax:

```sql
CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    ...
    CONSTRAINT constraint_name constraint_type (column)
);
```

- Example:

```sql
CREATE TABLE person (
    person_id SMALLINT UNSIGNED,
    fname VARCHAR(20),
    lname VARCHAR(20),
    gender ENUM('M','F'),
    birth_date DATE,
    street VARCHAR(30),
    city VARCHAR(20),
    state VARCHAR(20),
    country VARCHAR(20),
    postal_code VARCHAR(20),
    CONSTRAINT pk_person PRIMARY KEY (person_id)
);
```

## 4. Populating Tables

- INSERT syntax:

```sql
INSERT INTO table_name (column1, column2, ...)
VALUES (value1, value2, ...);
```

- Example:

```sql
INSERT INTO person
(person_id, fname, lname, gender, birth_date)
VALUES (null, 'William','Turner', 'M', '1972-05-27');
```

- Auto-increment for primary keys:

```sql
ALTER TABLE person MODIFY person_id SMALLINT UNSIGNED AUTO_INCREMENT;
```

## 5. Modifying Data

- UPDATE syntax:

```sql
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;
```

- DELETE syntax:

```sql
DELETE FROM table_name WHERE condition;
```

## 6. Common Mistakes

- Duplicate primary key:

```sql
INSERT INTO person (person_id, fname, lname, gender, birth_date)
VALUES (1, 'Charles','Fulton', 'M', '1968-01-15');
```

- Foreign key violation:

```sql
INSERT INTO favorite_food (person_id, food)
VALUES (999, 'lasagna');
```

## 7. Bank Schema

- Introduces tables like Customer, Account, Employee, Branch, etc.
- Viewing table structure:

```sql
DESCRIBE table_name;
```

or

```sql
SHOW COLUMNS FROM table_name;
```

## 8. MySQL-specific Features

- ENUM data type for check constraints:

```sql
gender ENUM('M','F')
```
